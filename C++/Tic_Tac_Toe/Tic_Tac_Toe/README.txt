>> User can play a standard game by typing 'y' at start.

>> Typing 'n' instead will prompt the player for inputs to play an m,n,k variation defined by the user.

>> After beginning a player can undo the last move made by typing in the number'0',
repeating this can revert the player back to the very first move.

>> Sources for reference were stack overflow and Geeks for Geeks mainly to help with translations,
from c# to c++ and just minor general syntax bumps found along the way.
